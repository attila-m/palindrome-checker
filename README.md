## Description

A simple webapp to check content for palindromes

## Prerequisites

* JDK 11 or later
* Maven 4.0 or later
* Docker
* Installed version of Postgres to test locally

## Start the app

To build the project, use
```bash
  ./mvnw clean package
```

To run the project, use
```bash
  docker-compose up
```
